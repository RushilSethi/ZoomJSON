# ZoomJSON

**A modern JSON explorer, formatter, and diff tool built for developers who work with JSON every day.**

🔗 **Live App:** [zoomjson.app](https://zoomjson.app)

---

## Why I Built This

Working with JSON is a daily reality for most developers — debugging API responses, reading config files, comparing payloads. Yet most online JSON tools feel outdated, cluttered with ads, or lack the features you actually need in one place.

ZoomJSON was born out of that frustration. I wanted a single, fast, beautifully designed tool that lets you **paste, explore, format, and compare JSON** without context-switching between multiple sites or installing heavy desktop apps.

## Why It's Helpful

- **Instant Formatting** — Paste raw or minified JSON and get clean, syntax-highlighted output immediately.
- **Interactive Explorer** — Navigate deeply nested structures with collapsible tree views, breadcrumb navigation, and keyboard shortcuts.
- **JSON Diff** — Compare two JSON objects side-by-side with additions, deletions, and modifications clearly highlighted.
- **Search** — Find keys or values across your entire JSON structure instantly.
- **Minimap** — Get a bird's-eye view of large JSON documents and jump to any section.
- **Multiple Themes** — Switch between light, dark, and specialty themes to match your workflow. Unlock the exclusive **Champion** theme by completing the hidden game on the home page!
- **Fullscreen Mode** — Expand input or viewer panels to fullscreen for distraction-free editing.
- **Mobile Friendly** — Fully responsive design that works on phones and tablets.
- **No Backend** — Everything runs client-side. Your data never leaves your browser.
- **Zero Ads** — Clean, focused interface with no distractions.

## Features at a Glance

| Feature | Description |
|---|---|
| 🧩 Tree View | Collapsible, interactive JSON tree with syntax highlighting |
| 🔍 Search | Real-time search across keys and values |
| 📊 Diff Tool | Side-by-side JSON comparison with change detection |
| 🗺️ Minimap | Overview navigation for large documents |
| 📌 Pinned Keys | Pin frequently accessed keys for quick reference |
| 🎨 Themes | Multiple color themes including an unlockable Champion theme |
| 📱 Responsive | Works seamlessly on desktop, tablet, and mobile |
| ⚡ Fast | Client-side only — no server round-trips |

## Tech Stack

- **React** + **TypeScript**
- **Vite** for blazing-fast development
- **Tailwind CSS** for styling
- **shadcn/ui** for accessible UI components
- **Framer Motion** for animations

## Getting Started

```bash
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd zoomjson

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:5173`.
